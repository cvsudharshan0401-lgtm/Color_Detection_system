# Color Detection System Using OpenCV and K-Means Clustering

## Project Overview

This project detects the dominant colors present in an image using OpenCV and K-Means Clustering. The system analyzes image pixels, identifies the top 5 dominant colors, displays their RGB values, assigns color names, and generates a color palette.

## Features

* Detects dominant colors from images
* Displays color names and RGB values
* Generates a color palette
* Uses K-Means Clustering
* Works in Google Colab

## Technologies Used

* Python
* OpenCV
* NumPy
* Google Colab
* K-Means Clustering

## Project Workflow

1. Upload an image.
2. Read the image using OpenCV.
3. Convert image pixels into a dataset.
4. Apply K-Means Clustering.
5. Extract dominant colors.
6. Display color names and RGB values.
7. Generate a visual color palette.
